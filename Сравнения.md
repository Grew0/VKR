

CPP vs JS

| Критерий                            | C++ + GUI LIB                                                | JS frontend application        |
| ----------------------------------- | ------------------------------------------------------------ | ------------------------------ |
| Метод чтение файлов                 | Из файловой системы                                          | Только через drag&drop         |
| Удобство пользователя               | Ниже                                                         | Выше                           |
| Скорость анализа хипа               | Выше за счет эффективности языка                             | Ниже                           |
| Простота разработки UI              | Ниже                                                         | Выше                           |
| Работа с большими данными           | Проще                                                        | Затруднена и ограничена языком |
| Кроссплатформенность                | Тяжелее, но возможно                                         | Легко                          |
| Возможность прямой интеграции с ark | Проще - одни язык                                            | Требуется nativa api           |
| Сложность компиляции                | Высокая - нужна GUI библиотека, а также кроссплатформенность | Отсутствует                    |
|                                     |                                                              |                                |
